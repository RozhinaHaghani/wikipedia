# Wikipedia Project
### List of technologies, libraries, and tools used in the project:

- Kotlin
- Material Design components
- Glide for image loading
- Recyclerview
- NestedScrollingView
- Toolbar
- Navigation Drawer
- Buttom Navigation
- Material CardView
